Simple Configuration

Configure everything in your Django settings or with simple commands

Settings Configuration

# settings.py
DJANGO_CLOUD = {
  "web": {"nodes": 2, "size": "small"},
  "redis": {"size": "medium"},
  "worker": {"nodes": 2},
  "domains": ["myapp.djangocloud.dev"]
}

Dynamic Scaling

# Scale on demand
$ django-cloud scale web --nodes 4
$ django-cloud scale worker --nodes 3

# Deploy updates
$ django-cloud deploy myapp
