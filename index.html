Deploy your Django app in seconds, just run

With a single command, skip the hassle and get your app running in the cloud effortlessly.

Django Cloud streamlines the process of building, deploying, and accessing a Django application with minimal effort. 

from django.http import JsonResponse
from django.urls import path
def hello_world(request):
    return JsonResponse({"message": "Hello World"})
urlpatterns = [path("", hello_world)]
django-cloud deploy
